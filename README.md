# BrawlBay
Brawlbay is a webapp that shows data for players of brawlhalla

Demo: https://brawlbay.com

## Getting Started

* git clone https://github.com/CCHaynes112/brawlbay.git
* docker-compose -f docker-compose-dev.yml up

## Built With

* React.js
* Django
* Docker
* Nginx
